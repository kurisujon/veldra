# User Roles

## Reviewer
- Conducts document review.
- Manages findings (Accepts, Resolves, Ignores).
- Generates Drafts (Affidavits, Explanation Letters).

## Supervisor
- Oversees cases and reviewers.
- Approves Drafts.
- Generates Final Export Packages.
- Manages Case assignments.

## Admin
- System configuration.
- User management and role assignment.
- Template management.
