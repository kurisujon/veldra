# Animation Guidelines

## Core Philosophy
Animations should be functional, subtle, and quick. They must never distract or slow down the user during long work sessions.

## Acceptable Animations
- **Hover States**: Fast, subtle color transitions (e.g., 150ms ease-in-out) on buttons and links.
- **Modals/Dialogs**: Quick fade-in and slight scale-up (e.g., 200ms ease-out) to bring focus without feeling sluggish.
- **Skeleton Loaders**: Gentle, slow pulsing opacity to indicate loading without causing motion sickness.

## Prohibited Animations
- Bouncing elements.
- Excessive slide-ins from off-screen.
- Parallax scrolling.
- Long, unskippable transitions.
